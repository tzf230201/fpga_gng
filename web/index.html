1 - Motivation & Problem

Micro-UAV swarms and field robots need on-board unsupervised adaptation. Cloud streaming is not reliable enough for payload-, power-, and link-constrained systems, so learning must run locally with bounded memory and deterministic timing.

  • Dense adjacency is O(N^2): 40 x 40 8-bit storage needs 1,600 B.
  • Floating-point units are expensive on small FPGA targets.
  • GNG has dynamic edge insert, age, and prune operations unlike static-lattice SOMs.
2 - Method Overview

The core contribution is a synthesizable VHDL engine on the Sipeed Tang Nano 9K: no soft-core CPU, no FPU, no PLL.

  1. Edge-cell encoding: upper-triangular age+1 cells.
  2. Q8/Q16 learning rates: one multiply plus shift.
  3. Fused NB_SCAN: decay, aging, movement, and pruning in one pass.
  4. Degree counters: O(Nmax) isolated-node detection.
3 - GNG Loop & Parameters

For each input sample, the engine finds winner s1 and runner-up s2, then updates graph topology incrementally.

4 - Edge-Cell Encoding

Edges are undirected, so only the upper triangle of the adjacency is stored. Each byte combines presence and age:

  • cell = 0: no edge
  • cell = age + 1: active edge
  • Over-age pruning becomes one compare against v > 51.
Ecells = Nmax(Nmax - 1) / 2 780 B versus 1,600 B dense edge storage
5 - Fixed-Point Arithmetic

Coordinates use signed integers in [-1000, 1000]. Distance computation produces a 35-bit unsigned result and maps cleanly to DSP resources.

d2 = dx^2 + dy^2 winner: 77 * delta >> 8 neighbor: 66 * delta >> 16 decay: err -= err >> 8

6-7 - Results

It fits, it learns, and it runs deterministically

Topologies are learned on FPGA and streamed through UART snapshots. The measured runtime stays near-identical across datasets because latency is dominated by bounded O(Nmax) BRAM scans.

8 - Takeaway & Future Work

Dynamic topology learning is feasible on a sub-10K-LUT FPGA

The poster's main message is that memory layout and arithmetic format are first-class design decisions. Bounded state plus incremental updates make this implementation a hardware substrate for continual learning at the edge.
