Growing Neural Gas on resource-constrained FPGA

A research implementation of Growing Neural Gas for Tang Nano 9K and NEORV32, using compact edge-cell graph encoding, integer arithmetic, UART visualization, and reproducible experiment logs.

Method

Bounded-memory GNG for small FPGA devices

The project maps the canonical GNG update rule into fixed-size hardware state: node BRAM stores coordinates, degree, activity, and error; edge BRAM stores only upper-triangular edge cells encoded as age+1 values.

Edge-cell graph encoding

Each edge uses one byte. A value of 0 means inactive, while values above 0 represent an active edge with age equal to value minus 1. This avoids a dense adjacency matrix and removes a separate active flag.

Integer and fixed-point arithmetic

Winner search, node movement, error decay, and edge aging are implemented with integer datapaths, Q-format learning rates, and shift-based approximations instead of a floating-point unit.

UART experiment interface

Dataset samples, node snapshots, edge snapshots, and profiling frames are sent through UART so the host can render the evolving topology and calculate scientific metrics from hardware output.
